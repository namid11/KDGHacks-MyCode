# 作品名

## ぼっちワーキングスペース

### 概要

ユーザの家のスペースなどをコワーキングスペースとして共有できるサービス。<br>
スマートキーを活用することで、特定のユーザのみを家に招待することができ、スマホを通して鍵を開閉できるような仕組みになっています。
